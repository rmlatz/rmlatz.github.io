GameGettr

This was the first full Ruby on Rails project I completed. It is designed to engage a user by asking them to answer a few simple questions (which correspond to parameters for making an API query) and returns a list of potential video game titles they may be interested in. This project contains a hand-rolled user authentication system and CRUD abilities of database persistence which allow a user to create, read, update and delete their own unique list of games to check out.

  • Technologies: Ruby on Rails, Postgresql, APIs

WildFire

WildFire was a group project carried out over a week with a team of 6. This app was designed to generate buzz for social events by masking the location of the event until sufficient use of a twitter hashtag had been achieved, using a variety of technologies. This project represented a deepdive into the agile development process, specifically the scrum methodology. Additionally, this project provided an opportunity to become very familiar with git workflow. Communication was incredibly important for this project and I was very lucky to have such an amazing team with which to work.

  • Technologies: Ruby on Rails, Postgresql, Devise, Figaro, JavaScript, Twitter for Rails, Google Maps
  • Team: Adam Barrett, Mary Hipp, Gabe Snyder, Mikael Davis, Thomas Weaver

VizVolt

This project was spent learning a new technology. I had barely used the D3 JavaScript library prior to this app. The idea of the app and of D3 is data visualization. By collecting data from the EIA API, I was able to use D3 to create a donut chart to visually display energy production in the United States and individual states by the fuel type used. This project is one I am very proud of, I feel I learned a lot through the process, not only just with the new technology I used, but with the process of learning a new technology in general. Additionally, this project is one I really want to continue to work on and expand into something that is really impressive in conveying information.

  • Technologies: Ruby on Rails, D3, APIs, Figaro, Postgresql
